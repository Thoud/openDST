# Digital Stress Test

The Digital Stress Test is a fork from [mbp-lab/openDST](https://github.com/mbp-lab/openDST) and is created with the JavaScript framework React.js in combination with [JATOS](https://www.jatos.org/) as a backend for study management.

The Digital Stress Test was originally developed in conjunction with [this paper](https://dx.doi.org/10.2196/32280). This fork updates the codebase to adapt the functionality for an upcoming study at the University of Hohenheim.

A presentation variant based on this code can not found publicly since the updated version will only be hosted internally and only for the duration of the upcoming study. However the original [repository](https://github.com/mbp-lab/openDST) contains a link to a presentation variant.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode on port [3000](http://localhost:3000) and includes hot module reloading.

### `npm run build`

Builds the app for production to the `build` folder.
It correctly bundles React in production mode and optimizes the build for the best performance.
